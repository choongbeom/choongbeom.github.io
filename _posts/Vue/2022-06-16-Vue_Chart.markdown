> Vue Chart 라이브러리

* Chart.js
  * [http://www.chartjs.org](http://www.chartjs.org) 
  * [https://github.com/apertureless/vue-chartjs](https://github.com/apertureless/vue-chartjs)
  * npm i vue-chartjs chart.js

<br>

* ApexCharts
  * [https://apexcharts.com/docs/vue-charts/](https://apexcharts.com/docs/vue-charts/)
  * npm install --save apexcharts
  * npm install --save vue-apexcharts

<br>

* Apache ECharts
  * [https://github.com/ecomfe/vue-echarts](https://github.com/ecomfe/vue-echarts)
  * npm install --save vue-echarts

<br>

* D3
  * [https://d3js.org/](https://d3js.org/)
  * npm install --save d3
  
<br>

* Google Charts: 
  * [https://developers.google.com/chart/interactive/docs/](https://developers.google.com/chart/interactive/docs/)
  * [https://github.com/devstark-com/vue-google-charts](https://github.com/devstark-com/vue-google-charts)
  * npm i vue-google-charts

<br>
